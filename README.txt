Linking to Stumpf, Reichskanzler
********************************

Die Reichskanzler vornehmlich des
X., XI. und XII.  Jahrhunderts.
Nebst einem Beitrage zu den
Regesten und zur Kritik der Kaiserurkunden

Concordance of document numbers ("Regesta")
and pages with links to the digitized
volume at the Bavarian State Library
<https://daten.digitale-sammlungen.de/~db/0000/bsb00001189/images/>

The main data is kept in ``data/stumpf_2_pages.json``.
The inverted list in ``data/stumpf_2_docs.json`` can be
created with the Python script in ``bin/stumpf_2_doc.py``.

If there are errors, always make sure to correct them in
``data/stumpf_2_pages.json``. For production purposes the
inverted JSON file might be more useful as it offers
a direct mapping from the document number to the corresponding
page url.
